EcmaScript 2015

EcmaScript 2015 (formerly called ES6) is the biggest feature release of JavaScript which introduced a lot of features used by every developer today. It introduced classes, arrow functions, new methods and more. It took a while before all major browsers fully supported them, but you can freely use them now
